# 首先簡單瞭解一下 tuple 是什麽東西

元組（tuple）是 Python 存儲容器之一，最大特點就是，它是 immutable，像其他 list、dictionary、set 這些，都是 mutable

當然，tuple 雖然是 immutable 類型，但其實跟 mutable 是沒有區別的，只是 CPython 在 tuple 實現上，我規定了不讓你修改，比如當你修改 tuple，我就給你抛異常

# Tuple 是怎麽發音？

通過網上瞭解到，Python 的創始人 Guido van Rossum，2011 年在 tweet 上鄭重説明 tuple 是怎么發音的

龜叔說，他是這樣念 tuple 的🤣

  1. 周一、三、五這三天，唸 too-pull
  1. 周二、四、六這三天，唸 tub-pull
  1. 星期日他不會談到 tuple，所以周日沒有 tuple 怎麼唸的問題

好了，回歸正題

# tuple 與 list 差別在哪

  1. tuple 創建時是用 () 表示，或者 tuple，如果調用 tuple () 來創建 tuple 效率並不高，應爲它需要執行 CALL_FUNCTION
  2. tuple 屬於 immutable，意味著，你不可以修改、增減 tuple 的值，因爲 tuple 的不可變特性，所以沒有 append、remove、pop 等可以變更值得操作
  3. 儅要建立只有一個 item 的 tuple 時，一定要加上逗號，否則不會建立 tuple

tuple 比 list 占用空間要更小

具體是怎麽計算大小的，以後有空再聊🐱‍👤

# tuple 有以下優點

  1. 因爲是 immutable，運行效率比 list 還要快
  2. 占用空間更小
  3. 可以作爲 dictionary 的 key 使用，因爲是不可變元素
